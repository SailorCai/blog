# docker 安装

再开始正式的内容之前，你需要一台云服务器，阿里云也好，腾讯云也好，哪个便宜买哪个就行，毕竟咱们目前只是用来学习，也不需要对外提供稳定的服务，不用考虑太多。我自己买的是腾讯云的 1 核 2G 加上新用户优惠那是相当便宜了，三年也才两百多。

买好服务器后按照下面的步骤安装 docker 就好了。

  • apt 升级
sudo apt-get update
  • 添加相关软件包
sudo apt-get install \
apt-transport-https \
ca-certificates \
curl \
software-properties-common
  • 下载软件包的合法性，需要添加软件源的 GPG 密钥
curl -fsSL https://mirrors.ustc.edu.cn/docker-ce/linux/ubuntu/gpg | sudo apt-key add -
  • source.list 中添加 Docker 软件源
sudo add-apt-repository \
  "deb [arch=amd64] https://mirrors.ustc.edu.cn/docker-ce/linux/ubuntu \
  $(lsb_release -cs) \
  stable"
  • 安装 Docker CE
sudo apt-get update
sudo apt-get install docker-ce
  • 启动 Docker CE
sudo systemctl enable docker sudo systemctl start docker
  • 建立 docker 用户组(附加)
sudo groupadd docker sudo usermod -aG docker $USER
  • Helloworld 测试
docker run hello-world

如能正常输出 hello world 就说明 docker 安装成功

  • 镜像加速

目前共有三个加速源

  1. Azure 中国镜像 https://dockerhub.azk8s.cn (opens new window)
  2. 阿里云加速器(需登录账号获取) (opens new window)
  3. 七牛云加速器 https://reg-mirror.qiniu.com (opens new window)

由于阿里云的镜像需要注册，咱们这里就把不需要注册的源添加上去：

在服务器的 /etc/docker/daemon.json 下添加下面代码

{
  "registry-mirrors": [
    "https://dockerhub.azk8s.cn",
    "https://reg-mirror.qiniu.com"
  ]
}
  • 最后重新载入配置文件并重启 docker
sudo systemctl daemon-reload
sudo systemctl restart docker
