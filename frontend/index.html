    # 小前端笔记

    这里是一些日常前端工作中的笔记，记下来的目的一是方便自己日后查看，二是希望能帮助到他人。

    如果文章内容有谬误或者您有更深的理解欢迎给我发邮件 sailorcai@163.com

    为了不让主页一片空白，我就聊聊自己这么些年做前端的感受吧！

    我自己从事前端行业已经有 5 年的时间了，从刚入行来到深圳进入一家创业公司切图写页面，到现在又进入了一家创业公司做前端；区别仅在于工资翻了好几倍，公司规模大一些。

    这些年间也进过腾讯外包，之后在一家做地产 saas 的公司 “明 x 云” 呆了两年，这些年经历过各家公司，在个人成长方面我自己有诸多的遗憾。

    个人在技术上的成长和智商、基础、运气、环境都有关系，就比如说自己身边有些同事对技术的嗅觉和理解能力真的比我高出一个 level，又或者另一些同事本身是计算机专业出身（我自己是非计算机相关专业，毕业后才转 iT 的），更或者有些朋友进入了非常好的团队，领导 nice，氛围又好，这些身边的朋友的成长速度非常快，有些早就进入大厂，有些已经是部门的技术负责人。

    这里我就当回忆一下这几年的职业生涯，希望一些经历能给看到的人一些启发。
